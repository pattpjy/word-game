## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:pattpjy/word-game.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run npm start
   ```sh
   npm run dev
   ```
4. Use the Link that is given in the Terminal & Open in Browser
   ```sh
   Example: http://localhost:3000/
   ```
